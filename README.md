## 📌실행 방법
### client 실행

1. **원격 저장소 복제**

```bash
$ git clone https://lab.ssafy.com/s08-webmobile1-sub2/S08P12D205.git
```

2. **프로젝트 폴더로 이동**

```bash
$ cd frontend
```

3. **필요한 node_modules 설치**

```bash
$ npm install
```

4. **개발 서버 실행**

```bash
$ npm start
```

<br />
